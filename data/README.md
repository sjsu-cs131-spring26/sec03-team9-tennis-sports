# Data Set Download Instructions
Datasets will __not__ be found in this repository.

## Manual Download
1. Visit the link to the dataset: https://www.kaggle.com/datasets/martj42/international-football-results-from-1872-to-2017/data?select=results.csv
2. Scroll past the description until you see the the Data Explorer portion.
3. Select _results.csv_ and click the download button.
